# Sudoku
9x9 board game, divided into 9 large squares and 9 squares within those. Objective: fill squares with numbers 1-9. Numbers can't repeat within squares, rows, or columns.
>Plan to have three difficulty levels. This will be done by adding different amount of initial values on the game board. E.g. 20 easy, 15 medium, 10 hard. 
>Find a way to create an interface for this game (not terminal).
